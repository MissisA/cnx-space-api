# 🚀 CNX-Space | Backend API Architecture

โปรเจกต์พัฒนาระบบหลังบ้าน (Backend) ประเภท RESTful API เพื่อศึกษาและประยุกต์ใช้โครงสร้างสถาปัตยกรรมระดับเซิร์ฟเวอร์ 

## ✨ ฟีเจอร์เด่น (Features)
- **OOP & Clean Controller:** ออกแบบโค้ดตามหลัก Object-Oriented Programming (OOP) และจัดระเบียบ Routing อย่างเป็นระบบ 
- **Dependency Injection:** ประยุกต์ใช้แนวคิด Dependency Injection เพื่อจัดการโค้ดให้เป็นโมดูล (Modularity) และง่ายต่อการบำรุงรักษา 

## 🛠️ เครื่องมือที่ใช้ (Tech Stack)
- **Backend:** C# 
- **Framework:** .NET 8 
- **Architecture:** Web API (RESTful)
